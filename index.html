Face Recognition System

A real-time face recognition system implementing advanced AI technologies. This project uses deep learning models to detect and recognize faces with high accuracy.

Technologies Used:
Python PyTorch OpenCV MTCNN InceptionResnetV1
Key Features:
  • Real-time face detection using MTCNN (Multi-task Cascaded Convolutional Networks)
  • Facial recognition with InceptionResnetV1 neural network from facenet_pytorch
  • Support for multiple faces in a single frame
  • Dynamic face database with easy addition of new faces
  • Confidence score display for recognition accuracy
  • Color-coded recognition feedback (green for known, red for unknown)
View on GitHub
Face Recognition System - Technical Details
How It Works:
  1. MTCNN detects faces in real-time from webcam feed
  2. Detected faces are processed through InceptionResnetV1 to create 512-dimensional embeddings
  3. Embeddings are compared with known faces using cosine similarity
  4. Recognition occurs when similarity exceeds 0.6 threshold
Installation & Usage: 
pip install -r requirements.txt
python face-recognition-opencv.py
Controls:
  • Q - Quit application
  • A - Add new person to database
  • S - Save face database
Privacy & Security:
  • All data stored locally in face_embeddings.pkl
  • No internet transmission of facial data
  • Embeddings only, not actual images
